= Enhancements

[role="_abstract"]
This release contains the following enhancements:

* You can now use the `PerformanceProfile` custom resource to configure worker nodes on machines equipped with Intel Sierra Forest or AMD Turin CPUs, which are supported when configured with a single NUMA domain (NPS=1).
